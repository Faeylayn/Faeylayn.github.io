Instagib

Live Github

Screenshot sharing app for videogame screenshots inspired by Instagram

Features

  • Uses a RESTful JSON API in Ruby on Rails to support the Backbone.js frontend
  • Overwrites the parse function to make fewer queries to the database
  • Uses Pusher to to implement realtime updates to the home feed.

Live Github

A classic arcade game rebuilt with Javascript, JQuery, and HTML5 Canvas.

Features

ActiveRecordLite

Github

An implementation of the Ruby on Rails ORM ActiveRecord

Features

  • Uses Ruby’s flexible structure to recreate ActiveRecord’s associations and database queries.
  • Creates a cascading relation that only calls a single query when all conditions have been set.
