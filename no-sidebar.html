Recent Work Described

Here are some of my recent projects:

AnonBot
Github
A Slack Bot that anonymizes slack messages written in Python
Features
  • Utilizes the Slack RTM API to gather the events firehose
  • Parses and selects direct messages to output to a specific slack channel
  • Uses 4 environment variables so the bot is reuseable for any slack or for multiple channels
JW PLayer Caption Editor
Blog Post
Caption Editor for JW Player
Features
  • Utilizes the Python RESTful API and HTML 5's Media API
  • Leverages dirty-checking with Angular.js to keep multiple components in sync
  • Incorporates Wavesurfer.js to add a synchronized waveform during playback
Battleship
Live Github
A classic board game rebuilt with Javascript and JQuery.

ActiveRecordLite
Github
An implementation of the Ruby on Rails ORM ActiveRecord
Features
